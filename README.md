# Elmo's Proxy Site Fetching System (EPSFS)
This github repo is meant for unblocked games sites/proxy sites who need a github repo that has cool content, without the hassle of getting permission from the owner.
# Rules
1. Don't get us blocked
2. Don't upload files called "index.html" "Cipher.html" "ESPFS.html" or any files with names that are already taken
3. Upload appropriate sites
4. Be chill
# Instructions
1. Choose a descriptive name for your file
2. Upload your file
